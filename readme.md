# Sistema de Auditoría de Inventario

Sistema web para el control y auditoría de inventarios, diseñado específicamente para auditores de inventario.

## Características

- ✅ Autenticación de usuarios por email y contraseña
- ✅ Carga de inventario teórico desde archivos Excel
- ✅ Escaneo de códigos de barras en tiempo real
- ✅ Visualización de hasta 1000 registros escaneados
- ✅ Edición y eliminación de ítems en caso de error
- ✅ Generación de informes PDF profesionales
- ✅ Resaltado de discrepancias con colores (rojo/amarillo)
- ✅ Historial de los últimos 5 inventarios
- ✅ Interfaz moderna y responsive
